# Changelog

All notable changes to **vr_mc_diagnostic** are recorded here.
The format follows [Keep a Changelog](https://keepachangelog.com/),
versions follow semantic-version-ish (`MAJOR.MINOR.PATCH`) with no
strict API guarantees pre-1.0 (none after either — this is an
operator-facing tool, not a library).

A user-facing copy of the latest release notes is also embedded in
the user guide (`docs/VinRobotics_MC_Diagnostic_User_Guide.md`,
appendix section "G. Changelog").
